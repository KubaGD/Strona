Gracze

Jak dołączyć w Brawl Stars

  1. Host w grze tworzy prywatny mecz i kopiuje kod (lub używa opcji create match).
  2. Host wkleja kod do pola "Wklej kod z Brawl Stars" i klika "Ustaw kod hosta".
  3. Inni gracze otwierają aplikację Brawl Stars i wklejają kod w opcji Join with code lub wybierają private match.
  4. Po wejściu do lobby robią screenshot i przesyłają go przez stronę, a następnie klikają "Potwierdzam".

Strona nie łączy się z serwerami Supercell — gracze muszą ręcznie wkleić kod w aplikacji. Upload screenshotów służy jako dowód dołączenia.
